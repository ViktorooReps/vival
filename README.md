# VIVAL

A simple commandline app for searching and looking up opensource vulnerabilities

# Installation

## Using Pip

```bash
  $ pip install vival
```
## Manual

```bash
  $ git clone https://github.com/ViktorooReps/vival
  $ cd vival
  $ python setup.py install
```

# Usage

`vival <executable> -t <path/tests.txt>`
```bash
$ vival
```
